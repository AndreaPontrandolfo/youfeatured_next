## Install

```sh
yarn install
```

## Usage

### Start the client dev server

```sh
cd client && yarn start
```

### Start the backend dev server

```sh
cd server && yarn dev
```

## Features

* Fullstack SPA
* Type-safety with typescript
* Highly scalalable through a complex infrastructure made of CI deployments with Docker and AWS

## Tech Stack

* Create-react-app
* Docker
* React w/Hooks
* Graphql
* Apollo
* Cypress
* Ramda
* Typescript
* Styled components
* PostgreSQL
* NGINX
